# 🤖 AI Test Case Generator (Excel Based)

## 📌 Overview
This project is a Python-based AI tool that automatically generates test cases from input requirements and exports them into an Excel file.

It helps QA engineers and testers reduce manual effort and improve test coverage.

---

## 🚀 Features
- Automatically generates test cases from input data
- Exports test cases into Excel format
- Saves time in manual QA work
- Improves test coverage and consistency

---

## 🛠️ Tech Stack
- Python
- Pandas
- OpenPyXL
- (Optional) AI/NLP logic

---

## 📂 Project Files
- `ai_testcase_excel.py` → Main script
- `test_cases.xlsx` → Generated output file
